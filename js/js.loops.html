JavaScript Loops


JavaScript loops allow you to run the same block of code multiple times. Common loop types include for, while, and do...while.

The for loop repeats a block of code a specific number of times. It has three parts: initialization, condition, and increment/decrement.

for (let i = 0; i < 5; i++) {
  console.log(i);
}

The while loop runs as long as the condition is true. It checks the condition before each iteration.

let i = 0;
while (i < 5) {
  console.log(i);
  i++;
}

The do...while loop runs the code block once before checking the condition. Then it repeats as long as the condition is true.

let i = 0;
do {
  console.log(i);
  i++;
} while (i < 5);

Loops are useful to iterate over arrays or repeat tasks efficiently without writing code multiple times.

In summary, JavaScript loops like for, while, and do...while help execute code repeatedly based on conditions or counts.
