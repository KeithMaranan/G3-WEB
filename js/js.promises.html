JavaScript Promises


Promises represent asynchronous operations that can complete in the future, either successfully or with an error.

A Promise is created with a constructor that takes a function with two parameters: resolve and reject.

let myPromise = new Promise((resolve, reject) => {
  let success = true;
  if(success) {
    resolve("Operation was successful");
  } else {
    reject("Operation failed");
  }
});

To handle the result, use .then() for success and .catch() for errors.

myPromise
  .then(result => console.log(result))
  .catch(error => console.log(error));

Promises make asynchronous code easier to read and manage compared to nested callbacks.

Promises represent future results of asynchronous operations, supporting chaining and error handling with then and catch methods.
