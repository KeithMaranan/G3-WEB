JavaScript DOM Manipulation


The DOM (Document Object Model) represents the structure of an HTML document. JavaScript can manipulate the DOM to change content, styles, and structure dynamically.

You can select HTML elements using methods like getElementById(), getElementsByClassName(), or querySelector().

let element = document.getElementById('myElement');
let items = document.getElementsByClassName('item');
let firstParagraph = document.querySelector('p');

To change the content inside an element, use innerHTML or textContent properties:

element.innerHTML = 'New content here';
element.textContent = 'Just text content';

You can also modify CSS styles of elements via the style property:

element.style.color = 'red';
element.style.backgroundColor = 'yellow';

To add or remove elements dynamically, use methods like createElement(), appendChild(), and removeChild():

let newDiv = document.createElement('div');
newDiv.textContent = 'Hello, I am new!';
document.body.appendChild(newDiv);

In summary, DOM manipulation lets you select, change, add, or remove HTML elements and their content or styles using JavaScript, making webpages interactive and dynamic.
