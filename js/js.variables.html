JavaScript Variables


Variables are containers used to store data values in JavaScript. They allow you to name and keep track of information that can change throughout the program.

You can declare a variable using var, let, or const. The main difference is in their scope and mutability.


let age = 25;
const name = "Alice";
var city = "New York";

let - is used for variables that can be changed later.

const - is used for constants values that cannot be reassigned.

var - is the older way to declare variables, and its usage is less common now due to its function scope.


Variable names must begin with a letter, underscore (_), or dollar sign ($). They cannot start with a number and are case-sensitive.


// Valid variable names
let _score = 100;
let $price = 9.99;
let userName = "John";

In summary, variables in JavaScript are declared with let, const, or var. Use let for changeable data, const for constants, and avoid var when possible for modern code.
