JavaScript Arrays


JavaScript arrays are used to store multiple values in a single variable. They can hold any type of data and allow you to organize collections of items.

Arrays are created using square brackets []. You can access items by their index, starting from 0.

let fruits = ["Apple", "Banana", "Cherry"];
console.log(fruits[0]); // Output: Apple

Common array methods include push() to add items, pop() to remove the last item, and length to get the number of elements.

fruits.push("Orange");
fruits.pop();
console.log(fruits.length); // Output: 3

You can loop through arrays using loops like for or forEach().

fruits.forEach(function(item) {
  console.log(item);
});

In summary, arrays in JavaScript store lists of data, can be accessed by index, and manipulated using built-in methods like push(), pop(), and forEach().
