JavaScript Conditions


JavaScript conditions allow you to perform different actions based on different values. They use statements like if, else if, and else.

The basic syntax of an if statement is:

if (condition) {
  // code to run if condition is true
}

You can add an else block to run code when the condition is false:

if (condition) {
  // code if true
} else {
  // code if false
}

To check multiple conditions, use else if:

if (condition1) {
  // code if condition1 true
} else if (condition2) {
  // code if condition2 true
} else {
  // code if none true
}

Conditions usually use comparison operators like ==, ===, !=, <, >, etc.

if (x === 10) {
  console.log("x is 10");
}

You can also use logical operators like && (and), || (or), and ! (not) to combine conditions.

if (x > 5 && x < 15) {
  console.log("x is between 6 and 14");
}

In summary, JavaScript conditions control the flow of code based on tests that are true or false, using if, else if, and else statements combined with comparison and logical operators.
