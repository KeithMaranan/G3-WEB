JavaScript Data Types


JavaScript has several built-in data types used to store different kinds of values. Understanding data types is essential to work effectively with variables.

The main data types in JavaScript include:


  • String: Text wrapped in quotes, e.g., "Hello" or 'World'.
  • Number: Numeric values, both integers and decimals, e.g., 42 or 3.14.
  • Boolean: Logical values, either true or false.
  • Undefined: A variable declared but not assigned a value.
  • Null: Represents an intentional absence of any object value.
  • Object: Collections of properties, used to store complex data.
  • Symbol: Unique and immutable data type, mostly used for object properties.

    • Examples of different data types:

    let message = "Hello, world!";
    let count = 10;
    let isActive = true;
    let result; // undefined
    let data = null;
    let person = {name: "Alice", age: 25};
