JavaScript Events


JavaScript events are actions or occurrences that happen in the browser, such as a user clicking a button, moving the mouse, or submitting a form. You can write code that reacts to these events.

The syntax to add an event handler directly in HTML is by using event attributes, such as onclick:

<button onclick="alert('Button clicked!')">Click me</button>

In JavaScript, you can also add event listeners using addEventListener() method:

document.getElementById('myBtn').addEventListener('click', function() {
  alert('Button clicked!');
});

Common event types include:

  • click – When an element is clicked
  • mouseover – When the mouse pointer moves over an element
  • mouseout – When the mouse pointer moves out of an element
  • keydown – When a key is pressed down
  • load – When the page or an image has finished loading

    • Events allow interactive web pages where user actions trigger JavaScript code.

    In summary, JavaScript events trigger code execution based on user actions or browser events. Use event attributes like onclick or addEventListener() to handle events.
