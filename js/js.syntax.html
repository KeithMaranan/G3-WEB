JavaScript Syntax


JavaScript syntax is the set of rules that define a correctly structured JavaScript program. It includes how you write statements, declare variables, and use functions.

JavaScript programs are made up of statements, which are executed line by line. Statements often end with a semicolon (;), although itâ€™s sometimes optional.

Here is the basic syntax to declare a variable:

let myVariable = 10;
const PI = 3.14;
var name = "John";

JavaScript uses functions to group code into reusable blocks:

function greet() {
  console.log("Hello!");
}

greet();

Comments in JavaScript help explain code and are ignored by the browser:

// This is a single-line comment

/*
  This is a multi-line comment
  that spans several lines
*/

JavaScript is case-sensitive, so myVariable and myvariable are different variables.

In summary, JavaScript syntax includes statements ending with semicolons, variable declarations using let, const, or var, functions for reusable code, and comments for code explanation.
