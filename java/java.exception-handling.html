Java Exception Handling


Exception handling manages runtime errors to maintain normal flow of execution.

It uses try, catch, finally, and throw keywords.

try {
  int result = 10 / 0;
} catch (ArithmeticException e) {
  System.out.println("Cannot divide by zero");
}

Summary: Exceptions provide a structured way to handle errors.
