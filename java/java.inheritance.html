Java Inheritance


Inheritance allows a class to acquire properties and behaviors of another class.

The class that inherits is the subclass; the class being inherited from is the superclass.

// Superclass

class Animal {
  void eat() { System.out.println("Eating..."); }
}

// Subclass
class Dog extends Animal {
  void bark() { System.out.println("Barking..."); }
}

Summary: Inheritance promotes code reusability and hierarchy.
