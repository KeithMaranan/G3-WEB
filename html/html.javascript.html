HTML JavaScript

JavaScript is a programming language that allows you to add interactivity, dynamic content, and behavior to your HTML web pages.
With JavaScript, you can create features like image sliders, form validation, animations, and interactive buttons.
Example
<button onclick="showMessage()">Click Me</button>

<script>
  function showMessage() {
    alert("Hello! This is JavaScript in action.");
  }
</script>
JavaScript code can be written inside the HTML file between <script> tags or in a separate .js file linked to the page.
JavaScript runs in the browser and is one of the core technologies of web development along with HTML and CSS.
