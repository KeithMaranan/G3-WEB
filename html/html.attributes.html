HTML Attributes


HTML attributes provide additional information about elements and are always placed in the start tag.

Key Facts About Attributes

  • Always written in the start tag
  • Usually come in name/value pairs like: name="value"
  • Modify the behavior or appearance of the element

    • Example

    <‎a href="https://www.example.com">Visit Example<‎/a>
    <‎img src="image.jpg" alt="A sample image">

    Common HTML Attributes

  • href - for links
  • src - for images
  • alt - alternative text
  • style - inline CSS
  • title - tooltip text

    • <‎p title="I'm a tooltip">Hover over this text<‎/p>

    Attribute Rules

    Attribute names are not case-sensitive, but lowercase is recommended. Attribute values should always be quoted.

    Bad Practice:

    <‎img src=image.jpg alt=sample>

    Good Practice:

    <‎img src="image.jpg" alt="Sample">
