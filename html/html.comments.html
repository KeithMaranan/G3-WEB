HTML Comments


HTML comments are used to insert notes into the HTML code that are not displayed in the browser. They are useful for explaining code, leaving reminders, or temporarily disabling parts of the HTML without deleting them.

The syntax for an HTML comment is <!-- comment text -->. Anything placed between <!-- and --> will be ignored by the browser and will not be shown on the page.

<!-- This is a comment -->
<p>This is a paragraph.</p>

You can place comments anywhere in the HTML code. They can be on their own line or next to HTML elements, and they will not affect the layout or appearance of the page.

<h1>This is a heading</h1> <!-- This is a heading comment -->
<p>This is some text.</p>

Comments can also be used to temporarily hide HTML elements from being displayed in the browser. This is useful for testing or when you do not want to permanently delete the code.

<!-- <p>This paragraph is hidden.</p> -->

HTML comments can span multiple lines by simply continuing the text until the closing -->. This is often done when writing detailed notes in the code.

<!--
This is a multi-line comment.
It can include several lines of text.
The browser will ignore all of it.
-->

In summary, HTML comments are created with <!-- and -->. They are not displayed in the browser and are used for adding explanations, reminders, or temporarily hiding code without removing it from the HTML file.
