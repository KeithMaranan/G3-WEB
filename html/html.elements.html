HTML Elements


An HTML element is defined by a start tag, some content, and an end tag.

HTML Elements

An HTML element is made up of:

  • a start tag
  • content
  • and an end tag

    • Sample Code

    <‎h1>My First Heading<‎ /h1>
    <‎p>My First paragraph.<‎ /p>
    Start Tag Content End Tag
    <‎h1> My First Heading <‎ /h1>
    <‎p> My first paragraph <‎ /p>
    <‎br> (none - empty) (none)

    Note: Some elements, like, are called empty elements, they don’t need a closing tag.

    Nested HTML Elements

    HTML elements can be nested, which means placing elements inside other elements.


    <‎!DOCTYPE‎ html>
    <‎head>
    <‎title>Sample Webpage<‎ /title>
    <‎ /head>
    <‎body>

    <‎h1>My first Heading<‎ /h1>
    <‎p>My first paragraph.<‎ /p>

    <‎ /body>
    <‎ /html>

    What’s Happening:

  • <‎html> is the root element (wraps the entire document)
  • <‎body> contains the visible page content
  • Inside the <‎body> :
  • <‎h1> = a heading
  • <‎p> = a paragraph


    • Never Skip End Tags

    While some browsers may still display content if you forget to close tags, don’t rely on it.

    Bad Practice:

    <‎p>This is a paragraph

    <‎p>This is a paragraph

    Empty HTML Elements

    Empty elements do not contain content and don’t have end tags.

    Example:

    <‎p> This is a <‎br> paragraph with a line break.<‎/p>

    HTML is Not Case Sensitive

    Tags like <‎P> and <‎p> are treated the same, but best practice is to always use lowercase tags.
