HTML Basic Examples


In this chapter, we’ll explore simple HTML examples to help you understand how it works.

Don't worry if you see unfamiliar tags, you’ll learn them as we go!

HTML Document Structure

Every HTML page should begin with a document type declaration: <‎!DOCTYPE html>

An HTML document starts with <‎html> and ends with <‎ /html>.

Everything visible on the page is placed between the <‎body> and <‎ /‎body> tags.

Sample Code

<‎!DOCTYPE‎ html>
<‎head>
<‎ /head>
<‎body>

<‎h1>This is a Heading<‎ /h1>
<‎p>This is a simple paragraph.<‎ /p>

<‎ /body>
<‎ /html>

<‎!DOCTYPE> Declaration

  • Declares the HTML version used.
  • Placed at the very top of your document.
  • Not case sensitive.
  • The declaration for HTML5 is: <‎!DOCTYPE html>

    • HTML Headings

    HTML headings are created using the tags <‎h1> to <‎h6>.

    <‎h1> is the biggest and most important, while <‎h6> is the smallest.


    <‎h1> This is heading 1 <‎ /h1>
    <‎h2> This is heading 2 <‎ /h2>
    <‎h3> This is heading 3 <‎ /h3>

    HTML Paragraphs

    Use the <‎p> tag to create paragraphs of text.


    <‎p>This is a paragpraph.<‎ /p>
    <‎p>This is another paragpraph.<‎ /p>

    HTML Links

    Use the <‎a> tag to create links.

    The href attribute specifies the destination URL.


    <‎a href="https://youtube.com/">Youtube<‎ /a>

    HTML Images

    The tag is used to display images.

    Important attributes include:

  • src (image source)
  • alt (alternative text)
  • width and height (size)

    • <‎img src="img.jpg " alt="image.com">

    How to View HTML Source

    Curious about how a website was made?

  • Press CTRL + U to view the full HTML source code.
  • Or right-click on a part of the page and choose Inspect to see the HTML and CSS used and even try editing them live.
