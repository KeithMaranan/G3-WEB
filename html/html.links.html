HTML Links


HTML links allow you to navigate from one web page to another, or to different parts of the same page. They are created using the <a> tag with the href attribute specifying the destination.

The syntax for an HTML link is <a href="url">link text</a>. The href attribute contains the URL or path to the destination. Clicking the link text will take the user to that location.

<a href="https://www.example.com">Visit Example</a>

You can also make a link open in a new browser tab or window by adding the target="_blank" attribute.

<a href="https://www.example.com" target="_blank">Open Example in New Tab</a>

Links can point to other pages on the same website using relative URLs, or to sections within the same page by linking to an element's id attribute using a hash symbol (#).

<a href="about.html">About Us</a>

<a href="#section2">Go to Section 2</a>

Images can also be used as links by placing the <img> tag inside an <a> element.

<a href="https://www.example.com"><img src="image.jpg" alt="Example"></a>

By default, browsers style links with an underline and blue color for unvisited links, purple for visited links, and red when active or clicked. You can customize this appearance using CSS.

In summary, HTML links are created with the <a> tag and the href attribute. You can open links in new tabs with target="_blank", use relative or absolute URLs, link to page sections with IDs, and style them with CSS.
