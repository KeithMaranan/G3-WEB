HTML Images


HTML images are used to display pictures on a web page. They are created with the <img> tag, which is an empty element with attributes that specify the image source and alternative text.

The basic syntax for displaying an image is <img src="url" alt="description">. The src attribute contains the path to the image file, and the alt attribute provides text to display if the image cannot load.

<img src="image.jpg" alt="Sample Image">

You can control the image size by adding the width and height attributes, either in pixels or percentages.

<img src="image.jpg" alt="Sample" width="300" height="200">

Images can also be linked to other pages by placing them inside an <a> element.

<a href="https://www.example.com"><img src="logo.png" alt="Example Logo"></a>

For responsive designs, you can use CSS to make images scale properly across devices.

img { max-width: 100%; height: auto; }

Always use meaningful alt text for accessibility and SEO purposes.

In summary, HTML images are displayed using the <img> tag with src and alt attributes. You can adjust their size, make them responsive with CSS, and even turn them into clickable links.
