HTML Lists


HTML lists are used to group related items together. They can be ordered (numbered) or unordered (bulleted), and each item in the list is placed inside a <li> tag.

An unordered list is created with the <ul> tag, and items are marked with bullets by default.

<ul>
  <li>Apple</li>
  <li>Banana</li>
  <li>Cherry</li>
</ul>

An ordered list is created with the <ol> tag, and items are numbered by default.

<ol>
  <li>First</li>
  <li>Second</li>
  <li>Third</li>
</ol>

You can change the numbering style in an ordered list using the type attribute.

<ol type="A">
  <li>Item A</li>
  <li>Item B</li>
</ol>

Lists can also be nested by placing one list inside another.

<ul>
  <li>Fruits
    <ul>
      <li>Apple</li>
      <li>Banana</li>
    </ul>
  </li>
</ul>

Definition lists are created with the <dl> tag, containing terms (<dt>) and descriptions (<dd>).

<dl>
  <dt>HTML</dt>
  <dd>A markup language for web pages.</dd>
</dl>

In summary, HTML lists can be unordered (<ul>), ordered (<ol>), or definition lists (<dl>). List items use the <li> tag, and you can style or nest lists for more complex structures.
