HTML Images


HTML images are used to display pictures on a webpage. They are created with the <img> tag, which is an empty element and does not have a closing tag.

The basic syntax for an image is <img src="url" alt="description">. The src attribute specifies the image path or URL, while the alt attribute provides alternative text for accessibility and for situations where the image cannot be displayed.

<img src="image.jpg" alt="A beautiful scenery">

You can control the size of an image using the width and height attributes, or through CSS for more flexibility.

<img src="image.jpg" alt="Scenery" width="400" height="300">

Images can also be linked by placing them inside an <a> tag.

<a href="https://www.example.com"><img src="logo.png" alt="Example Logo"></a>

For better page loading performance, you can use the loading="lazy" attribute to delay loading of images until they are needed.

<img src="large-photo.jpg" alt="Large Photo" loading="lazy">

In addition, you can use the title attribute to display a tooltip when the user hovers over the image.

<img src="flower.jpg" alt="Flower" title="This is a flower">

In summary, HTML images are displayed with the <img> tag using the src and alt attributes. You can resize them with width and height, link them with <a>, improve loading with loading="lazy", and add tooltips using the title attribute.
