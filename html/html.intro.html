HTML Introduction


HTML is the main language used to build and design web pages on the internet.

What is HTML?

  • HTML stands for Hyper Text Markup Language
  • It’s the standard language for creating websites.
  • HTML is used to define the structure and layout of a webpage.
  • It’s made up of different elements (or tags) that tell the browser how to show content.
  • Each element has a specific role, like marking something as a heading, paragraph, link, image, and more.

    • Sample Code

    <‎!DOCTYPE‎ html>
    <‎head>
    <‎title>HTML PAGE TITLE<‎/title>
    <‎/head>
    <‎body>

    <‎h1>This is a Heading 1<‎/h1>
    <‎p>This is a simple paragraph.<‎/p>

    <‎/body>
    <‎/html>

    What This Code Means

  • <‎!DOCTYPE html> – This tells the browser the document uses HTML5.

  • <‎html> – This is the main container for everything on the page.
  • <‎head> – This section holds hidden information like the title and metadata.
  • <‎title> – This sets the page title you see in the browser tab.
  • <‎body> – This contains everything visible on the page (text, images, buttons, etc.).
  • <‎h1> – This creates a main heading on the page.
  • <‎p> – This creates a paragraph of text.
