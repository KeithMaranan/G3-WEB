HTML Headings


HTML headings are used to define titles or subtitles on a webpage. They range from <h1> to <h6>.

HTML Headings

Headings help structure your content and indicate importance:

  • <h1> - Main heading (most important)
  • <h6> - Least important heading

    • Sample Code

    <h1>This is Heading 1</h1>
    <h2>This is Heading 2</h2>
    <h3>This is Heading 3</h3>

    Headings in HTML Structure

    Headings can be placed anywhere in the body of the HTML and are usually arranged in a hierarchy.

    <h1>Main Title</h1>
    <h2>Section Title</h2>
    <h3>Subsection</h3>

    Whatâ€™s Happening:

  • <h1> is the top-level heading
  • <h2> is a subheading
  • <h3> is a smaller subheading under h2


    • Do Not Skip Heading Levels

    Always use heading levels in order to maintain a clear structure for users and search engines.

    Bad Practice:

    <h1>Title</h1>

    <h4>Subtopic</h4>

    Good Practice:

    <h1>Title</h1>

    <h2>Subtopic</h2>

    HTML is Not Case Sensitive

    Tags like <H1> and <h1> work the same, but lowercase is recommended.
