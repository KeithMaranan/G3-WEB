HTML Paragraphs


A paragraph always starts on a new line, and is usually a block of text. In HTML, the <p> element defines a paragraph. Browsers automatically add some white space (a margin) before and after each paragraph.

In HTML, a paragraph will always begin on a new line, and it is displayed as a block-level element. You can create multiple paragraphs by adding more <p> tags. For example:

<p>This is a paragraph.</p>
<p>This is another paragraph.</p>

You cannot be sure exactly how HTML will be displayed because screen sizes and window sizes vary. Adding extra spaces or empty lines in the HTML source will not affect the way the content is displayed, as browsers automatically collapse multiple spaces and line breaks.

<p>
This paragraph
contains a lot of lines
in the source code,
but the browser
ignores it.
</p>

<p>
This paragraph contains a lot of spaces in the source code,
but the browser ignores it.
</p>

The <hr> element defines a thematic break in an HTML page, often displayed as a horizontal rule. It is an empty tag, meaning it has no end tag, and is used to separate content or indicate a change in topic.

<h1>This is heading 1</h1>
<p>This is some text.</p>
<hr>
<h2>This is heading 2</h2>
<p>This is some other text.</p>
<hr>

The <br> element defines a line break and is also an empty tag. It is useful when you need a new line without starting a new paragraph.

<p>This is<br>a paragraph<br>with line breaks.</p>

If you try to write a poem inside a normal paragraph, all lines will be displayed as a single continuous line because browsers ignore formatting spaces and line breaks in normal text.

<p>
My Bonnie lies over the ocean.

My Bonnie lies over the sea.

My Bonnie lies over the ocean.

Oh, bring back my Bonnie to me.
</p>

To preserve both spaces and line breaks exactly as written, use the <pre> element. Text inside a <pre> element is displayed in a fixed-width font and keeps all the spaces and formatting from the source code.

<pre>
My Bonnie lies over the ocean.

My Bonnie lies over the sea.

My Bonnie lies over the ocean.

Oh, bring back my Bonnie to me.
</pre>
