CSS External Styles


CSS external styles allow you to store your style rules in a separate file with a .css extension. This method makes it easier to maintain and update the design of a website, as all styles can be managed from one file instead of being written directly inside HTML.

To use an external CSS file, you create a separate file (e.g., style.css) and link it to your HTML file using the <link> element inside the <head> section.

<!DOCTYPE html>
<html>
<head>
<link rel="stylesheet" href="style.css">
</head>
<body>
<h1>Hello World</h1>
<p>This paragraph is styled from an external file.</p>
</body>
</html>

In your style.css file, you can define styles like this:

body {
  background-color: lightblue;
}

h1 {
  color: navy;
  text-align: center;
}

External styles are the best choice when working on large websites or multiple pages that share the same design. This way, you only need to edit one CSS file to update the style across all pages.

Always make sure the href path in the <link> element points to the correct location of your CSS file.

In summary, external stylesheets are linked to HTML using the <link> tag inside the <head> section, with the rel attribute set to "stylesheet" and the href attribute pointing to the CSS file.
