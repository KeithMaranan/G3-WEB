HTML ID

An HTML ID is an attribute that uniquely identifies a single HTML element on a page. Each ID must be unique within the document.
IDs are often used to apply specific CSS styles or to access an element with JavaScript for dynamic behavior.
Example
<style>
  #special { color: blue; font-weight: bold; }
</style>

<p id="special">This paragraph has a unique ID.</p>
In CSS, use a hash # before the ID name to style it. In JavaScript, target it with document.getElementById().
IDs are best for single, unique elements that need a specific style or behavior.
