HTML Style Attribute


The HTML style attribute is used to add styles to an element, such as color, font, size, and more. It allows you to apply CSS properties directly within an HTML tag.

The syntax for the style attribute is <tagname style="property:value;">, where the property is a CSS property and the value is the desired setting. For example, you can change the background color of a page using the background-color property inside the body tag.

<body style="background-color:powderblue;">
<h1>This is a heading</h1>
<p>This is a paragraph.</p>
</body>

You can also set background colors for individual elements, such as headings and paragraphs, by placing the style attribute directly in those tags.

<h1 style="background-color:powderblue;">This is a heading</h1>
<p style="background-color:tomato;">This is a paragraph.</p>

The color property changes the text color. The font-family property defines the typeface to be used, and the font-size property changes the size of the text.

<h1 style="color:blue;">This is a heading</h1>
<p style="color:red;">This is a paragraph.</p>

<h1 style="font-family:verdana;">This is a heading</h1>
<p style="font-family:courier;">This is a paragraph.</p>

<h1 style="font-size:300%;">This is a heading</h1>
<p style="font-size:160%;">This is a paragraph.</p>

The text-align property controls the horizontal alignment of text, such as centering a heading or paragraph.

<h1 style="text-align:center;">Centered Heading</h1>
<p style="text-align:center;">Centered paragraph.</p>

In summary, the style attribute lets you apply CSS directly within HTML elements. Use background-color for background colors, color for text colors, font-family for fonts, font-size for text sizes, and text-align for horizontal alignment.
