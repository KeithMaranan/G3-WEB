HTML Classes

An HTML class is an attribute used to assign one or more elements to a group for styling or scripting. Classes allow you to apply the same CSS styles or JavaScript functions to multiple elements at once.
A class can be used on any HTML element and can be reused as many times as needed across the page. This makes it a powerful tool for consistent styling.
Example
<style>
  .highlight { color: red; font-weight: bold; }
</style>

<p class="highlight">This text is styled with a class.</p>
<h3 class="highlight">This heading uses the same class.</h3>
Use a dot . before the class name in CSS to style it. In JavaScript, you can target it with document.getElementsByClassName() or querySelectorAll().
Classes make it easy to apply consistent styles and behaviors to multiple elements without repeating code.
