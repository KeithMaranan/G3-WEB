HTML Div

The HTML div element is a container used to group together HTML elements so they can be styled or manipulated as a unit. It does not add any visual change by itself but becomes useful when combined with CSS or JavaScript.
A div can hold text, images, tables, or other elements, and it is commonly used to structure web pages into sections. Divs are block-level elements, meaning they start on a new line and take up the full width available.
Example
<div style="background-color: lightblue; padding: 20px;">
  <h3>This is inside a div</h3>
  <p>You can add multiple elements inside a div.</p>
</div>
Divs are essential in creating layouts, as they help divide a webpage into sections for better styling control and design flexibility.
Use a div when you want to group elements together for styling or scripting purposes.
