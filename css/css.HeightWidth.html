CSS Height & Width

CSS height and width properties are used to set the size of an elementâ€™s content area. You can specify them in units like pixels (px), percentages (%), em, rem, and more.

The syntax is simple:

selector {
  width: value;
  height: value;
}

For example, setting a div to be 300 pixels wide and 150 pixels tall:

div {
  width: 300px;
  height: 150px;
}

You can also use percentages to make the size relative to the parent element:

div {
  width: 50%;
  height: 100px;
}

When height or width is not set, elements size automatically based on their content.

In summary, use width and height to control element dimensions in CSS, using units like px or % depending on your design needs.
