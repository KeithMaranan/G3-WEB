CSS Padding

CSS padding creates space between an element's content and its border.
The padding property controls the inner spacing of an element. You can set it for all sides or individually with padding-top, padding-right, padding-bottom, and padding-left.

Example
h1 {
  padding: 20px;
}

p {
  padding-left: 15px;
  padding-right: 15px;
}
In this example, the h1 element has equal padding on all sides, while the p element has extra space only on the left and right sides.
