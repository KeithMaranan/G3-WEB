CSS Syntax

CSS syntax is the set of rules that define how CSS is written. It consists of selectors and declarations that specify how HTML elements should be styled.
A CSS rule has two main parts: a selector, and a declaration block. The selector points to the HTML element you want to style. The declaration block contains one or more declarations separated by semicolons.

Example
selector {
  property: value;
  property: value;
}
In this example, the selector targets an element, and the properties inside the curly braces define the styles. Each property has a value, and each declaration ends with a semicolon.
