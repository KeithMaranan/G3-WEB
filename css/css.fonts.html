CSS Fonts

CSS font properties control the appearance of text fonts, including font family, size, weight, and style.

Common font properties:

selector {
  font-family: value;
  font-size: value;
  font-weight: value;
  font-style: value;
}

Example:

h1 {
  font-family: Arial, sans-serif;
  font-size: 24px;
  font-weight: bold;
  font-style: italic;
}

This will make h1 headings use Arial font, 24px size, bold and italic style.

Use CSS font properties to customize font appearance such as typeface, size, weight, and style.
