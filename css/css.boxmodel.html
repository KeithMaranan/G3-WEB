CSS Box Model

The CSS box model describes the rectangular boxes generated for elements. It consists of margins, borders, padding, and the actual content area.

The box model has four parts:

  • Content: The actual content like text or images.
  • Padding: Space between content and border.
  • Border: The border surrounding the padding and content.
  • Margin: Space outside the border, separating elements.

You can set the size of each part using CSS properties like width, height, padding, border, and margin.


Example
div {
  width: 300px;
  height: 150px;
  padding: 20px;
  border: 5px solid black;
  margin: 10px;
}

This will create a box with:

  • Content area of 300px wide and 150px high
  • Padding of 20px inside the border
  • A solid 5px black border
  • 10px margin outside the border

In summary, the CSS box model controls the space and border around elements. Understanding it is key to controlling layout and design.
