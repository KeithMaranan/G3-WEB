CSS Introduction

CSS (Cascading Style Sheets) describes how HTML elements are to be displayed on screen, paper, or in other media.
CSS saves a lot of work by controlling the layout of multiple web pages all at once. It can be added to HTML in three ways: inline, internal, and external.
Example
<style>
  body {
    background-color: lightblue;
  }
  h1 {
    color: white;
    text-align: center;
  }
</style>
This CSS changes the background color of the page and centers the heading text in white color.
CSS is essential for making web pages visually appealing and improving user experience.
