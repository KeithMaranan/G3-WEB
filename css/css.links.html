CSS Links

CSS allows you to style links in different states such as normal, hover, visited, and active.

The link pseudo-classes are:

  • a:link — unvisited links
  • a:hover — mouse over links
  • a:visited — visited links
  • a:active — active/clicked links

Example of styling links:

a:link { color: blue; }
a:hover { color: red; }
a:visited { color: purple; }
a:active { color: orange; }

This changes link colors based on user interaction.

Use link pseudo-classes to customize how links look in different states for better user experience.
