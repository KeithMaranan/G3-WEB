CSS Text

CSS text properties allow you to style and format the text inside HTML elements.

Common text properties include color, text-align, text-decoration, and text-transform.

selector {
  color: value;
  text-align: value;
  text-decoration: value;
  text-transform: value;
}

Example:

p {
  color: blue;
  text-align: center;
  text-decoration: underline;
  text-transform: uppercase;
}

This will style paragraphs with blue, center-aligned, underlined, uppercase text.

Use CSS text properties to control the color, alignment, decoration, and case of your text.
