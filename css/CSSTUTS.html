CSS Home

CSS (Cascading Style Sheets) is used to style and layout web pages. It can control the color, font, spacing, and positioning of elements,

making websites look more attractive and user-friendly.

CSS allows you to separate content from design, which means you can update the look of a website without changing its HTML structure. It can be applied to HTML elements in three ways: inline, internal, and external.

Example
<style>
  body {
    background-color: lightblue;
  }
</style>

<h1>Welcome to CSS</h1>
In this example, the background color of the page is set to light blue using CSS. This is an internal CSS example because the styles are written inside a <style> tag in the HTML document.
