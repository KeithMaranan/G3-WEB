CSS Borders

CSS borders define the boundary around an element and can be styled with different widths, colors, and patterns.
The border property allows you to set the border's width, style, and color. You can also set each side individually using border-top, border-right, border-bottom, and border-left.

Example
p {
  border: 2px solid black;
}

div {
  border-top: 4px dashed red;
  border-bottom: 4px dotted blue;
}
In this example, the p element has a solid black border, while the div element has different styles for the top and bottom borders.
